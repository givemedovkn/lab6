# Задание 1

### Текст задачи
Необходимо разработать прибор для диагностики автомобиля. Если у автомобиля сломан двигатель или нарушено давление в колесах, то он считается сломанным. 
Также необходимо разработать усовершенствованный прибор, который при наличии поломки может указать, какая именно поломка произошла, и дать рекомендации. 
Также он может принимать давление в каждом колесе и проводить повторный анализ на нарушение (Стандартное давление 2,5 бар).

### Алгоритм решения

Базовый класс: 
В классе есть 2 логических поля, которые описывают состояние двигателя и колес.
Есть 3 конструктора: 
1 конструкор по умолчанию устанавливает значение состояния двигателя и колес - не сломано.
2 конструктор по параметру устанавливает переданные значения состояний двигаетля и колес.
3 конструктор копирует значения состояний двигателя и колес у выбранной машины.
Аскесор для состояния двигаетеля: передает значение состояния двигателя или устанавливает полученное значение.
Аскесор для состояния колес: передает значение состояния двигателя или устанавливает полученное значение.
Метод опрелеления состояния машины (сломана ли): если двигатель или колеса сломаны, значит машина сломана. 

Дочерний класс: 
В классе имеется массив давления каждого колеса.
Есть 3 конструкора:
1 конструкор по умолчанию устанавливает значение состояния колес 2.5
2 конструктор по параметру принимает массив значений и устанавливает его для массива давления каждого колеса.
3 конструктор по параметру принимает массив значений и устанавливает его для массива давления каждого колеса и принимает значение двигателя и устанавливает его значением состояния двигателя.
Метод оценки давления колес: если хотя бы одно колесо не равно 2,5, то давление в колесах нарушено.
Метод рекомендации: если метод определения состояния из базового класса вернул false, значит машина в порядке. иначе проверяется состояние двигателя (если true - выдача рекомендаци) и
давление в колесах (если true, то проверяется каждое колесо и нарушенные колеса выводятся).

# Задание 2

### Текст задачи
LineSegment
double x, double y (координаты начала и конца отрезка на координатной прямой)
Определить пересекаются ли заданные отрезки. Результат должен быть типа bool.

### Алгоритм решения

Класс LineSegment:
В классе есть два вещественных поля: координаты начала (x) и конца (y) отрезка на координатной прямой.
Конструктор по умолчанию: создает единичный отрезок [0; 1].
С параметрами: инициализирует координаты с проверкой (начало не может быть больше конца).
Перегрузка операторов сравнения < и >: выполняют проверку пересечения двух отрезков (< возвращает true, если пересекаются; > возвращает false, если пересекаются).

# Задание 3

### Текст задачи
Унарные операции:
! вычислить длину диапазона, результатом должно быть вещественное число
++ расширить отрезок на 1 вправо и влево
Операции приведения типа:
int (неявная) – результатом является целая часть координаты х;
double(явная) – результатом является координата y.
Бинарные операции:
- LineSegment целое число (левосторонняя операция, уменьшается координата х);
- целое число LineSegment (правосторонняя операция, уменьшается координата y);
< LineSegment d – результат равен true, если отрезки пересекаются, и false–в противном случае.

### Алгоритм решения

Класс LineSegment:
В классе есть два вещественных поля: координаты начала (x) и конца (y) отрезка на координатной прямой.
Конструктор по умолчанию: создает единичный отрезок [0; 1].
С параметрами: инициализирует координаты с проверкой (начало не может быть больше конца).
Перегрузка операций:
Унарный оператор !: вычисляет длину отрезка как разность между концом и началом.
Оператор инкремента ++: расширяет отрезок влево и вправо на 1.
Арифметические операторы -: 
отрезок - число: сдвигает начало отрезка влево на указанное число.
число - отрезок: сдвигает конец отрезка влево на указанное число.
Операторы сравнения < и >: выполняют проверку пересечения двух отрезков (< возвращает true, если пересекаются; > возвращает false, если пересекаются).
Операторы преобразования типов:
Неявный int: позволяет получить координату начала x при приведении к int.
Явный (double): позволяет получить координату конца y при приведении к double.
